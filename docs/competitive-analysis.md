# Competitive Analysis

## Alternatives

| Alternative | Strength | Weakness / Gap | Case Mirror Differentiation |
|---|---|---|---|
| Peer practice | Human judgment and realism | Availability varies; feedback quality is inconsistent | Always available structured rehearsal and report |
| Coach or TA feedback | High-quality expert feedback | Limited time; often happens late | Pre-coaching readiness check before expert sessions |
| ChatGPT / Claude / Gemini prompts | Flexible, fast question generation | Requires manual prompt design; no guided workflow or report | Purpose-built case setup, Q&A, scoring, and report flow |
| Presentation coaching tools | Delivery and speech feedback | Usually not case/rubric/recommendation aware | Combines business-case logic with observable delivery signals |
| Slide review tools | Useful for deck polish | Does not pressure-test Q&A | Focuses on judge-style challenge questions and follow-ups |
| Manual rubric checklist | Clear criteria | No adaptive practice or answer feedback | Converts rubric into active rehearsal prompts |

## Saturation Risk

The broad AI productivity and presentation-feedback market is crowded. Case Mirror reduces saturation risk by focusing on a specific user journey: student case-competition rehearsal. The differentiation is not simply "AI-powered feedback"; it is the combination of case prompt, rubric, recommendation, Q&A answers, and observable delivery evidence inside one guided workflow.

## Positioning

Case Mirror should be positioned as:

> A case competition rehearsal mirror that helps teams see the weak parts of their recommendation before judges do.

It should not be positioned as:

- an official judge simulator
- an emotion detector
- a personality or confidence scorer
- a replacement for coaches
- a generic presentation app

## What To Validate Next

- Whether coaches find the readiness report credible.
- Which report sections are most actionable.
- Whether Teachable Machine gesture signals add useful coaching value or distract from content feedback.
